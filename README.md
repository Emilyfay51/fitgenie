# FitGenie

FitGenie is a responsive fitness planning web app that helps users generate personalized workout routines based on their goals, experience level, training schedule, workout location, and intermittent fasting style. It combines workout generation with hydration guidance, calorie estimates, recovery tips, progress tracking, and saved plans to create a more complete fitness planning experience.

## Features

- Generate workout routines based on fitness goals
- Select experience level and number of workout days per week
- Choose home workouts, gym workouts, or a flexible training setup
- Adjust workout duration for 30, 45, or 60 minute sessions
- Include intermittent fasting guidance
- View hydration targets based on activity level and workout duration
- See estimated BMI and calorie guidance
- Get warm-up suggestions, meal timing tips, and rest-day advice
- Track completed exercises with progress checkboxes
- Save multiple workout plans and reload favorites
- Toggle between light and dark mode
- Use the app on desktop or mobile

## Built With

- HTML
- CSS
- JavaScript
- localStorage

## How It Works

FitGenie takes user inputs such as goal, experience level, days per week, workout location, workout duration, body measurements, activity level, and intermittent fasting preference. Based on these selections, it generates a structured weekly workout plan and displays additional coaching insights like hydration goals, calorie estimates, meal timing recommendations, and progress tracking.

## Why I Built It

I built FitGenie to practice creating a real-world web app that uses user input, logic, and dynamic rendering to deliver personalized results. I wanted to create something more polished than a basic form project and build an app that feels useful, interactive, and portfolio-ready.

## Run Locally

1. Clone or download the repository
2. Open the project folder in VS Code
3. Open `index.html` in your browser

Or run a simple local server:

```powershell
python -m http.server 8000
```
