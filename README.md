# FitGenie

FitGenie is a polished beginner-friendly workout planner built with plain HTML, CSS, and JavaScript.

## Features

- goal, experience, training-day, and fasting inputs
- home, gym, or flexible workout mode
- 30, 45, or 60 minute workout planning
- hydration goal calculator
- BMI and calorie estimate cards
- warm-up, meal timing, and rest-day coaching
- clean day cards with exercise descriptions
- progress tracking with checkboxes
- save multiple plans and reload favorites
- light and dark theme toggle
- responsive layout for desktop and mobile

## Run It

You can open `index.html` directly in a browser.
